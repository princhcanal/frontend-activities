Exercise 1: Centering Stuff

One of the first things beginner front-end web developers struggle with, myself included, is just centering things on the page. In this exercise, create a div with a height of 20rem and width of 80%. Choose a background color for your div. Inside the div, create an h1. Center the div itself and also center the h1 inside the div. Both the div and the h1 should be cented horizontally and vertically. The black background represents your whole page and the white rectangle represents your div.

The Foundations of Web Development

HTML5

HTML stands for Hyper Text Markup language and describes what elements are on the page. You can say that HTML is the noun of a website.

CSS3

CSS stands for Cascading Style Sheets and describes the style of the elements on a website. You can say that CSS is the adjective of a website.

JavaScript

JavaScript is a programming language and it helps makes a website more dynamic. It describes the behavior of the elements on a website. You can say that JavaScript is the verb of a website.

Exercise 4: Background Images

An img tag already exists in HTML, so why use background-image? Well, as you probably already know, the img tag is a self closing tag. This means that we cannot put content inside of the image. This is one of the benefits of using the background property on a div in CSS. In this exercise, you will create a div with text inside of your choosing then use CSS to set the background image of that div. The image I used can be found here.
